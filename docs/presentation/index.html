CORE CHALLENGE

核心难点：Chunk语义完整性

问题

MineRU按物理布局切分文档，可能打断语义边界

# Chunk 1
本产品提供三年
# Chunk 2
质保服务，包含免费维修
语义被打断，检索匹配度降低

解决方案：Parent-Child Chunking

小chunk用于检索，大chunk提供上下文

Parent Chunk (提供上下文)
本产品提供三年质保服务，包含免费维修。响应时间：24小时内到达现场。
检索时匹配Child，返回Parent
Child Chunks (精确匹配)
本产品提供 三年质保服务 24小时到达
