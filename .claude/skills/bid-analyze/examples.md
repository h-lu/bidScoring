# 投标分析 Skill 使用示例

## 示例1：基本分析

**用户输入：**
```
分析项目 abc123 的投标文档
```

**Skill执行：**
1. 调用 `list_available_versions(project_id="abc123")` 获取文档列表
2. 自动识别投标人名称
3. 执行6维度分析
4. 生成报告

**输出：**
```markdown
# 投标分析报告

## 1. 概览
- 项目：XX设备采购项目
- 投标方数量：3家
- 分析时间：2025-02-10

## 2. 综合排名
| 排名 | 投标方 | 综合得分 | 风险等级 |
|------|--------|----------|----------|
| 1 | 上海悦晟生物科技 | 78.5 | medium |
| 2 | 北京某科技公司 | 72.3 | medium |
| 3 | 广东某贸易公司 | 58.1 | high |
```

## 示例2：指定投标人名称

**用户输入：**
```
分析这些投标文档：
- version-001: 悦晟生物
- version-002: 东方科技
- version-003: 南方实业
```

**Skill执行：**
1. 使用提供的投标人名称
2. 对3个version_id执行分析
3. 生成对比报告

---

## 示例3：只分析特定维度

**用户输入：**
```
对比这几个投标方的质保和交付能力
```

**Skill执行：**
1. 设置 `dimensions=["warranty", "delivery"]`
2. 只分析这两个维度
3. 生成维度对比表

---

## 示例4：生成标注PDF

**用户输入：**
```
分析项目abc123，生成标注PDF
```

**Skill执行：**
1. 执行完整分析
2. 为每个投标方调用 `highlight_pdf()`
3. 返回PDF下载链接

**输出：**
```markdown
## 标注PDF

| 投标方 | 标注PDF | 说明 |
|--------|---------|------|
| 悦晟生物 | [下载](https://...) | 风险(红) 优势(绿) |
| 东方科技 | [下载](https://...) | 风险(红) 优势(绿) |
```

---

## 示例5：交互式追问

**用户输入：**
```
悦晟生物的质保期具体是多少？在文档哪里？
```

**Skill执行：**
1. 调用 `extract_key_value(version_id, ["质保期", "保修期"], ["年", "个月"])`
2. 调用 `get_chunk_with_context(chunk_id)` 获取原文位置

**输出：**
```markdown
根据文档分析，悦晟生物的质保承诺如下：

- **质保期：5年** (第3页，第5段)
- **免费维修：质保期内免费** (第3页，第6段)
- **响应时间：24小时内到场** (第4页，第2段)

原文位置：document_version=xxx, page_idx=3, chunk_id=yyy
```

---

## 示例6：对比特定条款

**用户输入：**
```
对比各家的培训方案
```

**Skill执行：**
1. 调用 `compare_across_versions(version_ids, "培训技术指导")`
2. 汇总对比结果

**输出：**
```markdown
## 培训方案对比

| 投标方 | 培训天数 | 培训内容 | 师资承诺 | 评分 |
|--------|----------|----------|----------|------|
| 悦晟生物 | 5天 | 操作+维护 | 原厂认证 | 85 |
| 东方科技 | 3天 | 仅操作 | 未明确 | 60 |
| 南方实业 | 未明确 | 未明确 | 未明确 | 30 |
```
